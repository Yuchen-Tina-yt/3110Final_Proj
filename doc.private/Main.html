Module Main

module Main: sig .. end

This module contains all the functions that plays and runs the game. Nothing inside the module should be exposed to the user.

exception Illegal
val print_strlist : string list -> unit

print_strlist is the function that prints the string list to the console

val welcome : Gamestate.t -> unit

welcome state prints the welcome message and the player's name, and also the place name that the player is currently on

val winornot : Gamestate.t -> bool

winornot player is true if the player wins, and false otherwise.

val roll : Gamestate.t -> unit

roll st mutates the state and the player after moving the player to the place in the state that corresponds to the number of the rolled die.

val play_round : Gamestate.t -> unit

play_round st is the function that lets the player play a single round.

val lottery : Gamestate.t -> unit

lottery st is the lottery system in the game that randomly give the player money, let the player lose money, move the player forward a random step, or let the player participate in the chance card system.

val explore : Gamestate.t -> unit

explore st allows the player explore the state and make commands. The function mutates the state and the player according to the parsed user input commands

val choose_rent_or_battle : Gamestate.t -> unit

choose_rent_or_battle st is the function that lets the current player choose to pay the rent or participate in battle

val play_game : Gamestate.t -> unit

play_game state is the function that allows the players to play the game

val play : Gamestate.t -> unit

play state is the function that plays the game. It is called in main to activate the game.

val main : unit -> unit

main () prompts for the game to play, then starts it.
